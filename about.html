關於 PurePDF

PurePDF 是一個致力於提供安全、快速、隱私優先的線上 PDF 工具平台。

在數位化時代，我們經常需要處理含有個人隱私或商業機密的文件。市面上許多的線上 PDF 工具雖然方便，但往往需要將您的檔案上傳至遠端伺服器進行處理。這不僅受限於網路速度，更存在著資料外洩的潛在風險。

PurePDF 的誕生正是為了解決這個問題。我們利用現代瀏覽器強大的運算能力（WebAssembly 與 JavaScript），將原本需要伺服器處理的工作，全部移至您的本地端瀏覽器完成。

隱私權政策

核心承諾：檔案不離手

當您使用 PurePDF 時，您選擇的檔案永遠不會離開您的裝置。所有的運算過程（讀取、重組、轉檔）皆是在您的電腦或手機記憶體中進行。我們沒有後端資料庫，也不會儲存您的任何文件副本。

1. 資料收集

我們不收集使用者的文件內容、檔名或任何個人識別資料。本網站託管於 GitHub Pages，可能會由 GitHub 收集標準的伺服器存取日誌（如 IP 位址），這屬於 GitHub 的服務條款範疇。

2. Cookie 與追蹤

PurePDF 本身不使用 Cookie 來追蹤您的個人行為。我們僅可能使用本地儲存（Local Storage）來暫存您的操作偏好（如選擇的解析度設定），這些資料僅存於您的瀏覽器中。

3. 外部連結

本網站可能包含通往其他網站（如作者個人網站或 GitHub 儲存庫）的連結。我們無法控制這些外部網站的隱私權實務。

技術與開源致謝

PurePDF 是建立在巨人的肩膀上。我們使用了以下優秀的開源專案來實現純前端處理：

  • PDF.js (Mozilla) - 用於渲染 PDF 預覽
  • pdf-lib - 用於修改與建立 PDF 結構
  • React - 用於建構互動介面
  • Tailwind CSS - 用於介面樣式設計
